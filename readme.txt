== Description ==

Displays the current months sales, transactions and more on the dashboard. Requires WP Shopping Cart plugin by Instinct Entertainment. http://www.instinct.co.nz/e-commerce/

Supports all languages that the WP Shopping Cart provides.

Installation

1. Unzip the archive of the plugin.
2. Upload folder `wp-e-commerce-rightnow` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

See my blog for technical updates at http://www.sebs-studio.com/tag/e-commerce/
